# 경동 사이트 클론코딩

> HTML, CSS, JavaScript를 활용해 경동 공식 웹사이트의 메인 페이지를 클론코딩한 포트폴리오 프로젝트입니다.

## 프로젝트 소개

본 프로젝트는 경동 공식 사이트의 레이아웃과 주요 UI 구성을 참고하여 제작한 클론코딩 작업물입니다.  
웹 페이지의 구조를 직접 분석하고, 메인 비주얼 영역과 스와이프 인터랙션을 구현하며 정적인 마크업을 실제 웹 화면처럼 구성하는 데 중점을 두었습니다.

- 원본 사이트: http://www.kyungdong.co.kr/
- 작업 유형: 웹사이트 클론코딩
- 제작 목적: 취업 포트폴리오 및 퍼블리싱 역량 향상
- 사용 기술: HTML, CSS, JavaScript

## 주요 기능

### 메인 비주얼 슬라이드

- 메인 영역에 스와이프 기능 적용
- 이전/다음 슬라이드 전환 기능 구현
- 사용자 인터랙션에 따라 자연스럽게 콘텐츠가 변경되도록 구성

### 웹 페이지 구조 구현

- Header, Visual, Contents, Footer 영역 구성
- 원본 사이트의 전체적인 레이아웃과 분위기를 참고하여 구현
- HTML 시맨틱 구조를 고려한 마크업 작성

### CSS 스타일링

- 섹션별 레이아웃 구성
- 이미지, 텍스트, 버튼 요소 스타일링
- 사이트 분위기에 맞는 컬러와 간격 적용

### JavaScript 인터랙션

- 스와이프 및 슬라이드 동작 구현
- 사용자 이벤트에 따른 화면 전환 처리
- 동적인 UI 동작을 JavaScript로 제어

## 사용 기술

| 구분    | 기술          |
| ------- | ------------- |
| Markup  | HTML5         |
| Style   | CSS3          |
| Script  | JavaScript    |
| UI 기능 | Swipe / Slide |

## 폴더 구조

```bash
kyungdong-clone/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
└── README.md
```

> 실제 폴더명과 파일명은 업로드한 프로젝트 구조에 맞게 수정할 수 있습니다.

## 실행 방법

1. 저장소를 클론합니다.

```bash
git clone https://github.com/사용자아이디/저장소명.git
```

2. 프로젝트 폴더로 이동합니다.

```bash
cd 저장소명
```

3. `index.html` 파일을 브라우저에서 실행합니다.

또는 VS Code의 Live Server 확장 기능을 사용해 실행할 수 있습니다.

## 작업 과정에서 중점적으로 본 부분

- 원본 사이트의 레이아웃을 분석하고 HTML 구조로 재구성
- CSS를 활용한 섹션별 화면 배치 및 디자인 구현
- JavaScript를 이용한 스와이프 기능 적용
- 사용자 경험을 고려한 자연스러운 슬라이드 인터랙션 구현
- 포트폴리오용으로 코드 구조와 파일 관리를 정리

## 배운 점

이번 작업을 통해 실제 기업 사이트의 UI 구성을 분석하고, 이를 HTML, CSS, JavaScript로 직접 구현하는 과정을 경험했습니다.  
특히 메인 비주얼 영역의 스와이프 기능을 적용하면서 단순한 정적 페이지가 아닌 사용자와 상호작용하는 웹 페이지 제작 흐름을 익힐 수 있었습니다.
